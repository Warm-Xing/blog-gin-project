# 个人博客系统后端

这是一个基于Go语言、Gin框架和GORM库开发的个人博客系统后端，实现了用户认证、文章管理和评论功能。

## 功能特点

- 用户注册和登录（JWT认证）
- 文章的创建、读取、更新和删除（CRUD）
- 评论的创建、读取和删除
- 权限控制（只有作者可以修改/删除自己的文章和评论）
- 分页查询支持

## 技术栈

- 编程语言：Go 1.16+
- Web框架：Gin
- ORM：GORM



## 项目总结

本项目基于Gin框架和GORM库实现了一个功能完善的个人博客系统后端。通过这个项目，我们学习了如何：

1. 初始化Go项目并管理依赖
2. 设计和实现RESTful API
3. 使用GORM进行数据库操作
4. 实现JWT认证和授权
5. 组织项目结构以提高可维护性
6. 处理错误和记录日志

这个系统虽然简单，但包含了Web应用开发中的许多核心概念和最佳实践。你可以基于这个项目继续扩展功能，如添加文件上传、富文本编辑、标签和分类、搜索功能等。

## 可能的扩展方向

1. 添加更完善的用户角色和权限管理
2. 实现文章的标签和分类功能
3. 添加全文搜索功能
4. 集成第三方登录（如GitHub、Google）
5. 实现API限流和缓存
